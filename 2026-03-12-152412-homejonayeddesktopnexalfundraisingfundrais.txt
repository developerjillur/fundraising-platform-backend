  ---
  Recommended AWS Architecture

  Tier 1: Compute — ECS Fargate (both frontend & backend)

  Why Fargate over EC2/EKS/Lambda:
  - Both apps already have Dockerfiles — zero refactoring needed
  - No servers to patch or manage (serverless containers)
  - Pay only for what you use (vCPU + memory per second)
  - SSE long-lived connections work fine (unlike Lambda's 15min timeout)
  - Scales automatically based on demand
  - Cost: ~$15-30/month for light traffic (0.25 vCPU, 0.5GB each)

  Tier 2: Database — Amazon RDS MySQL (Free Tier eligible)

  - db.t3.micro or db.t4g.micro — free for 12 months
  - Automated backups, Multi-AZ option for production
  - No database management overhead
  - Cost: ~$0 (free tier) → ~$15-25/month after

  Tier 3: File Storage — Amazon S3 + CloudFront

  - Replace local uploads/ with S3 bucket (minor code change in
  photo.service.ts)
  - CloudFront CDN for fast delivery of photos/screenshots globally
  - Virtually unlimited storage
  - Cost: ~$1-5/month for moderate usage

  Tier 4: Networking & CDN — Application Load Balancer + CloudFront

  - ALB routes traffic to Fargate tasks (health checks, SSL termination)
  - CloudFront in front of the frontend for caching static assets
  - Cost: ~$16-20/month (ALB minimum)

  Tier 5: DNS & SSL — Route 53 + ACM

  - Route 53 for domain management
  - ACM provides free SSL certificates
  - Cost: ~$0.50/month (hosted zone) + $0 for SSL

  Tier 6: CI/CD — GitHub Actions + ECR

  - GitHub Actions: Free for public repos, 2000 min/month for private
  - ECR (Elastic Container Registry): Store Docker images
  - Pipeline: Push → Build Docker images → Push to ECR → Deploy to ECS
  - Separate workflows for backend/ and frontend/
  - Cost: ~$0-1/month

  Tier 7: Secrets & Config — AWS Systems Manager Parameter Store

  - Store all your env vars (Stripe keys, JWT secret, DB creds, API keys)
  - Free for standard parameters
  - ECS tasks pull secrets at launch
  - Cost: $0

  Tier 8: Monitoring — CloudWatch

  - Container logs, metrics, alarms
  - Basic monitoring included free
  - Cost: ~$0-3/month
